I always wanted to start a blog.

It began as a WordPress site.

Then I realized… I could build one myself.

---

## Design & Tailwind

I started designing layouts.

Mobile + desktop.

Then came Tailwind.

And with it:

- npm commands  
- “command not found” errors  
- massive node_modules  

I built desktop first…

Then learned Tailwind is mobile-first.

So I rebuilt everything.

Lesson learned.

---

## React & Components

Instead of repeating code, I learned to build reusable components.

Example:

Blog cards = same structure  
Different content = props  

Props = arguments  
State = memory  

---

## Routing

Using React Router:

- no page reloads  
- smoother navigation  

Single-page apps feel fast because of this.

---

## Storage & Deployment

Initially:

- blogs stored locally  

Later options:

- database

- better formatting
